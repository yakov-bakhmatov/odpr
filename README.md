odpr
====

Ложные корреляции по открытым данным Пермского края

Загрузить данные

`lein run load-data`

Подготовить данные

`lein run prepare-data`

Вычислить попарные корреляции

`lein run calculate`

Отфильтровать корреляции и данные

`lein run prepare-correlation`

Запустить веб-сервер

`lein run serve`
